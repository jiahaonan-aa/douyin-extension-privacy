Douyin 文本横幅检测器 - 插件隐私政策

适用范围：本插件仅供公司内部人员使用，不面向公众。

本插件收集哪些信息？

插件不会收集、传输或存储任何个人信息或浏览历史。所有处理行为均在用户浏览器本地完成。

使用权限说明

  • 插件会扫描网页中是否包含 douyin.com / iesdouyin.com 的链接。
  • 插件不会与任何外部服务器通信，亦不包含埋点、统计、广告等行为。
  • 插件使用 broad host 权限，仅为检测方便所设。

数据传输和第三方

插件不使用任何第三方库、CDN 或网络资源；所有功能均在本地执行。

Douyin Link Banner Detector - Privacy Policy

Scope: This extension is intended for internal company use only, not for public distribution.

What data is collected?

No personal data, browsing history, or any other user information is collected, transmitted, or stored.

Permissions

  • The extension only checks for presence of douyin.com / iesdouyin.com links in the current page.
  • No network requests or remote communications are made.
  • Broad host permissions are used solely for detection functionality.

Data Handling

The extension is fully self-contained and does not interact with third parties or remote servers.
